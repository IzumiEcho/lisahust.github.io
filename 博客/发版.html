发版流程

  1. appcenter-edith发版

  2. gateway:default:prod

  3. start Manual Execution

  4. 选择自己的发版号（一般要等gitlab编译）

发版注意

  1. 发版到一半需要继续点击，是否发版
  2. 我们一共有八台实例，发版过程中，会四个四个的转移到新一版中
  3. 在发版的过程中，实例会由红变绿
  4. gitlab显示绿勾不一定表示编译成功，需要进去cat里看看
